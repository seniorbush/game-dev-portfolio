UFO - Rail Shooter

3D Rail Shooter.

Based on a tutorial from GameDev.tv, this project gave me the chance to experiment and implement a variety of differnet features. Made use of Unity's terrain tools to scuplt the environment, paint textures and add trees etc. I used the timeline to add animations for the enemy ships and objects. The player controls for this game were alot more specific, having to take into considersation the pitch, yaw and roll of the players ship.

Throughly enjoyed this project and learned many more features within Unity and wrote numerous different scripts for the game.
